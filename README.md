Installation
-------------

```sh
$ git clone https://github.com/pocke/brainfuck-web
$ cd brainfuck-web
$ npm i
$ gulp tsd
$ gulp ts
$ gulp browserify
```


Testing
-------

```sh
$ npm test
```
